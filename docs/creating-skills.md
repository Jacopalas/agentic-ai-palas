# Creating Skills

See the official documentation: **[Agent Skills Overview](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview)**

## Contributing a Skill

1. Create `.claude/skills/your-skill/SKILL.md`
2. Update the skills table in `README.md`
3. Submit a pull request

## Update CLAUDE.md

If you create a new skill, remember to update CLAUDE.md so it knows about the new skill.
